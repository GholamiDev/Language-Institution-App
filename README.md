 # Language Institution Platform 🎓

یک پلتفرم جامع آموزش زبان که امکان مدیریت تعاملات بین دانشجو، استاد و ادمین را فراهم می‌کند. این اپلیکیشن با معماری Full-stack طراحی شده و فرآیندهای ثبت‌نام، ایجاد دوره، تایید ادمین و خرید دوره را به صورت یکپارچه مدیریت می‌کند.

## 🚀 تکنولوژی‌های استفاده شده

### Frontend
- **React.js**: کتابخانه اصلی برای رابط کاربری.
- **Redux**: مدیریت متمرکز State اپلیکیشن.
- **Responsive Design**: طراحی کاملاً واکنش‌گرا برای دسکتاپ و موبایل.
- **Dark/Light Mode**: پشتیبانی از تم‌های تیره و روشن.

### Backend
- **Django**: فریم‌ورک قدرتمند برای مدیریت دیتابیس و منطقِ سمت سرور (API).

## ✨ ویژگی‌های کلیدی (Core Features)

### برای اساتید
- **ثبت‌نام چندمرحله‌ای (Multi-step Registration)**: فرآیند ثبت‌نام دقیق.
- **پنل اختصاصی**: پس از تایید ادمین، اساتید می‌توانند دوره‌های خود را ایجاد و مدیریت کنند.

### برای دانشجوها
- **مشاهده پروفایل اساتید**: بررسی رزومه و تخصص اساتید قبل از انتخاب.
- **فیلترینگ پیشرفته**: جستجو و فیلتر دوره‌ها بر اساس قیمت، زبان و جدیدترین‌ها.
- **خرید آسان**: دسترسی به دوره‌های تایید شده توسط ادمین.

### مدیریت
- **سیستم تایید (Approval Workflow)**: کنترل ادمین بر ثبت‌نام اساتید و تایید دوره‌های ایجاد شده برای حفظ کیفیت آموزشی.
- **بخش مقالات**: ارائه محتوای آموزشی مکمل.
